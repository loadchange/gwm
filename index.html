Quick Start ​

Installation ​

bash
npm install gwm

Basic Usage ​

javascript
// Import the library
import gwm from 'gwm';

// Create a watermark with default options
gwm.creation();

// Create a watermark with custom options
gwm.creation({
  txt: 'Confidential Document',
  fontSize: 14,
  color: '#ff0000',
  alpha: 0.2,
  angle: -20
});
